Cautionary Tales for Children and Kew Gardens

Members of this group are Erin Mooney, Yuying Jin, Aidan Ray, Sudarshan Veludandi, and Nicholas Wright. Our group is under the supervision of Professor Beshero-Bondar.

This group was brought together for a text encoding assignment. We all took interest in the stories in these books, so we were selected to work together to properly code the stories!
